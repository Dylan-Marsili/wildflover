# Anime Backgrounds

Bu klasöre yüksek kaliteli anime background görselleri eklenecek.

## Önerilen Kaynaklar

Pinterest'ten aşağıdaki anime serilerinden 4K kalitede background görselleri:
- Jujutsu Kaisen
- Hunter x Hunter (HxH)
- Demon Slayer
- Attack on Titan
- Your Name

## Dosya Formatı

- Format: JPG veya PNG
- Çözünürlük: Minimum 1920x1080 (4K önerilir)
- Dosya adı: anime-bg.jpg

## Kullanım

Görseli `anime-bg.jpg` olarak bu klasöre kaydedin.
Uygulama otomatik olarak background olarak kullanacaktır.
