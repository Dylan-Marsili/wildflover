# Application Icons

Icon dosyaları bu klasörde bulunmalıdır.

## Gerekli Dosyalar

- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)

## Oluşturma

Tauri CLI ile otomatik oluşturabilirsiniz:
```bash
npm run tauri icon path/to/icon.png
```

Veya online araçlar kullanabilirsiniz:
- https://icon.kitchen/
- https://www.icoconverter.com/
